# Rick & Morty Characters

**Студент:** Червякова Ольга Дмитриевна  
**Группа:** Б9124-09.09.03пикд3

**Тема приложения:** Приложение для просмотра персонажей Rick & Morty.

**Описание:** Приложение позволяет смотреть список персонажей, фильтровать их по статусу, искать по имени, открывать экран деталей персонажа.

**Функции приложения:**
- Отображение списка персонажей
- Фильтр по статусу (All / Alive / Dead / Unknown)
- Поиск по имени персонажа
- Просмотр деталей персонажа
- Навигация между экранами (Jetpack Navigation)

**API:** [Rick and Morty API](https://rickandmortyapi.com/)
